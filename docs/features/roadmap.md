# Roadmap de Funcionalidades Futuras

Este documento centraliza as próximas evoluções planejadas para a plataforma ImobIA, organizadas por impacto e complexidade.

## 1. Módulo Financeiro: Automação de Repasses
- **Objetivo**: Liquidar automaticamente a parte do proprietário após o recebimento do aluguel.
- **Funcionalidades**:
    - Dashboard de "Contas a Pagar" (Repasses).
    - Integração com APIs bancárias para transferências (Pix/TED).
    - Geração automática de recibos de repasse.

## 2. Comunicação: Notificações Inteligentes
- **Objetivo**: Reduzir a inadimplência e manter partes interessadas informadas.
- **Funcionalidades**:
    - Régua de cobrança automática via WhatsApp (5 dias antes, dia do vencimento, 3 dias após).
    - Notificação de "Proposta Recebida" para proprietários.
    - Alertas de "Documentação Aprovada" para inquilinos.

## 3. Jurídico: Assinatura Digital Nativa
- **Objetivo**: Eliminar a necessidade de plataformas externas (DocuSign/Clicksign).
- **Funcionalidades**:
    - Workflow de coleta de assinaturas dentro da plataforma.
    - Verificação de identidade via Selfie/Documento.
    - Hash de segurança e registro de log de assinatura.

## 4. Análise: Score de Crédito Automatizado
- **Objetivo**: Agilizar a tomada de decisão em propostas de aluguel.
- **Funcionalidades**:
    - Integração direta com birôs de crédito (Serasa/Boa Vista).
    - Sugestão automática de aprovação baseada em regras de comprometimento de renda.

## 5. SaaS: Gestão de Planos e Assinaturas
- **Objetivo**: Monetizar a plataforma por módulos.
- **Funcionalidades**:
    - Configuração de limites por plano (ex: Plano Base até 50 imóveis).
    - Habilitação dinâmica de módulos na Sidebar conforme o plano.
    - Checkout nativo para upgrade de conta.
